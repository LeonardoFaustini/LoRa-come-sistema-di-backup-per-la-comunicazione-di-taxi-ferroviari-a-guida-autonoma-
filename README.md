# LoRa-come-sistema-di-backup-per-la-comunicazione-di-taxi-ferroviari-a-guida-autonoma
In questa repository riporto tutto il materiale che concerne la mia tesi triennale (2025/26). Basata sul lavoro di tirocinio presso RailEvo esplora lo sviluppo prototipale e analisi parametrica della tecnologia LoRa come sistema di backup per la comunicazione di dati telemetrici dei veicoli Railevo.
